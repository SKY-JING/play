# 一、登陆
```
$ npm adduser //注册账号
Username: YOUR_USER_NAME
Password: YOUR_PASSWORD
Email: YOUR_EMAIL@domain.com
```

# 二、发布
```
// 公共包发布
npm publish
// 私有包发布
npm publish --access=public
```

# 三、更新版本号
```
npm version patch
```
